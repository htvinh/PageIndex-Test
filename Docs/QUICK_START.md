# Quick Start Guide (Local Mode)

Get up and running with the PageIndex Testing Application (Local Edition).

## 🚀 Prerequisites

- **Python 3.8+**
- **Ollama** or **OMLX** server running.

## 📦 Setup

1. **Navigate to project:** `cd PageIndex-Test`
2. **Configure:** Edit `config.json` to match your local Ollama/OMLX settings.
3. **Launch:** Run `./scripts/launch.sh`.

## 🎮 Usage

1. **Document Upload:** In the **Document Analysis** tab, upload your PDFs.
2. **Aggregation:** Select documents from the list and click "Aggregate Trees".
3. **Analyze:** Enter your query in the text area and click "Perform Analysis".

The system will retrieve relevant segments (text + visual images) and generate a comprehensive answer.
